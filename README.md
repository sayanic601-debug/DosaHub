# DosaHub

A modern South Indian Food Ordering Website.

## Features
- User Login & Signup
- Browse South Indian Menu
- Add to Cart
- Order Food
- Responsive Design

## Tech Stack
- Next.js
- React.js
- Tailwind CSS
- Node.js
- MongoDB

## Installation
```bash
npm install
npm run dev
```
